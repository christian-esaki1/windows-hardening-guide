# 🤝 Comment Contribuer

Merci de votre intérêt pour contribuer à ce projet !

## Signaler un Bug

- Ouvrez une [Issue GitHub](https://github.com/christian-esaki1/windows-hardening-guide/issues)
- Décrivez le problème clairement
- Incluez votre version de Windows et PowerShell
- Fournissez les étapes pour reproduire le bug

## Proposer une Fonctionnalité

- Ouvrez une issue avec le tag `enhancement`
- Décrivez la fonctionnalité et son utilité
- Discutez de l'implémentation

## Soumettre du Code

1. Forkez le projet
2. Créez une branche (`git checkout -b feature/ma-fonctionnalite`)
3. Commitez vos changements (`git commit -m 'Ajout de...'`)
4. Pushez (`git push origin feature/ma-fonctionnalite`)
5. Ouvrez une Pull Request

## Standards PowerShell

- Utilisez les verbes approuvés (Get-, Set-, Test-, etc.)
- Commentez votre code
- Incluez la gestion d'erreurs (try-catch)
- Fournissez des exemples d'utilisation

## Questions ?

N'hésitez pas à ouvrir une issue pour toute question !
